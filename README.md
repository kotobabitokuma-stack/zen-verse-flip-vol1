# Kotonoha Bright Key (KBK) - Zen Verse Flip vol.1

このプロジェクトは、Pi Network上で展開される「KBK経済圏」の第一弾アプリ **Zen Verse Flip vol.1** です。

## 🚀 プロジェクトの目的
「自分の好きを大切にし、他人の好きを尊重し、みんなで平和な場所を作る」というKBKの3つのルールに基づき、メタバース内での建築やNFT展示を支援するプラットフォームを目指しています。

## 🛠️ 主な機能
- **Pi Network 決済統合**: 3.0 Piでのサポート決済機能。
- **NFTフレーム自動リサイズ**: 展示するNFTのサイズに合わせた額縁の自動調整設計。
- **マーケットプレイス構想**: 未来のロードマップに販売機能を追加予定。

## 💻 開発環境
- **Framework**: React
- **Hosting**: Vercel
- **Device**: Mac Ventura 13.7.8 / i5 2.3GHz / 16GB RAM
